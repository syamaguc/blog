## 序文

[![Magic the Gathering](https://i.gyazo.com/1cf6e3f81282f4dcafce05f2812aede0.jpg)](https://gyazo.com/1cf6e3f81282f4dcafce05f2812aede0)
マジックザギャザリング(以下MTG)の再録禁止カードについて考えたことを書いてみる。

MTGには公式が今後、同名や機能的同一なカードを絶対に再録(再印刷)しないと公式に名言している、再録禁止カードというものが存在する。

MTGの初期セットの各レアが何枚印刷されたか？

```
Alpha　               1100
Beta　                3200
Unlimited　           18500
Revised　             289000
Legends　             19500
Arabian Nights　      20500
Antiquities　         31000
The Dark              128000
Fallen Empire         744000
Ice Age               202000
Homelands　           413000
Alliances　           326086
Mirage　              242424
Visions　             240000
Wheatherlight　       240000
Tempest　             242424
Stronghold　          272727
Exodus　              272727
Urza's Saga　         363636
Urza's Legacy　       303030
Urza's Destiney       303030
```

これはすなわち各セットの各再録禁止カードが世界にこれだけの枚数存在するということである。(=これだけの枚数しか存在しない)

具体的には以下のようになる。

```

Black Lotus(Alpha)                1100
volcanic island(beta)             3200
Black Lotus(unlimited)            18500
Underground Sea(Revised)          289000
The Tabernacle at Pendrell Vale   19500
Mishra's Workshop                 31000
Lion's Eye Diamond                242424
Mox Diamond                       272727
Gaea's Cradle                     363636
...
...
...
etc

```

重要なことはヴィンテージのポケモンカードと違い、MTGの再録禁止カードは永久にプレイされる需要があることです。(エターナルフォーマット、統率者)

ポケモンカードなどはスタン落ち(発売から一定期間経過したカードは大会で使用出来なくなる)の概念があるため、スタン落ち後はコレクションとして厳重に保管される。

そのため、美品状態で残っているカードの割合はMTGより格段に多いと推定される。(かつ、今後も美品の数がプレイによる損耗等で激減することはない)

ChatGPTによる推定によれば、MTGの再録禁止各カードの総供給に対して現時点での割合は、

| 状態             | 推定 | レンジ   |
| ---------------- | ---- | -------- |
| 未開封           | 4%   | 2 - 6%   |
| NM               | 28%  | 22 - 35% |
| SP               | 27%  | 22 - 30% |
| MP - HP          | 26%  | 20 - 30% |
| HP以下(消滅含む) | 15%  | 10 - 20% |

またデュアルランド各種やGaea's Cradleのようにプレイ需要が多いかつデッキに4枚必要なカードに関しては、

| 状態             | 推定 | レンジ   |
| ---------------- | ---- | -------- |
| 未開封           | 2%   | 2 - 3%   |
| NM               | 22%  | 18 - 25% |
| SP               | 33%  | 30 - 35% |
| MP - HP          | 29%  | 27 - 33% |
| HP以下(消滅含む) | 14%  | 12 - 18% |

というような推論が出ている。

実際に知名度や流動性の高い、Underground Sea(Revised)で枚数を推定すると、

| 状態             | 推定    |
| ---------------- | ------- |
| 未開封           | 5780枚  |
| NM               | 63580枚 |
| SP               | 95370枚 |
| MP - HP          | 83810枚 |
| HP以下(消滅含む) | 40460枚 |

となる。
そして恐ろしいことは、

- 供給は固定されている(もう増えない)
- 今後もプレイによる消耗や人的管理ミスによって美品の枚数は減っていく
- 運営の決算や公式発表によるとMTGのプレイ人口は近年増加傾向である(需要は増えている)

ということ。

## 市場規模

有名なUnderground Sea(Revised版)について考えてみる。このカードは価格と流動性的に最も規模を把握するのに適切なカードだと思われる。

[![Underground Sea](https://i.gyazo.com/2988984728f2c421eb5d206ad9e59e52.jpg)](https://gyazo.com/2988984728f2c421eb5d206ad9e59e52)

執筆時点(2026年4月)の晴れる屋の価格を見てみると、

```
NM: 289000 * 22% * 220000円    = 115億円
SP: 289000 * 33% * 176000円    = 130億円
MP-HP: 289000 * 29% * 150000円 = 100億円

```

Revised版のUnderground Seaの市場希望だけで約350億円がある。(ちなみにNMの在庫はゼロ、SPの在庫は1のみ)

比較対象として、現在高騰中のポケモンカードの代表的シンボルであるマリオピカチュウと比較してみると、PSA10の鑑定枚数が2000枚強でPSA10の販売価格が540万円前後なので、100億円強くらいと考えられる。(たぶんポケモンカードでは単体で最も市場規模の大きなカード)

Reserved Listには572枚のカードがあります。(うち300種以上に25ドル以上の値がついています)

仮にその300枚の総供給枚数の平均をざっくり200000枚とし、価格を10000円とすると、それだけで約6000億円の市場規模があることになります。

そして日本で最も在庫量のある晴れる屋で、高額な再録禁止カードの在庫はほとんどありません。

仮に運営が公式に名言している再録禁止の約束を反故にした場合、これだけの価値が毀損されるので、少なくとも数千億円規模の損害賠償の訴訟に発展します。

これはWotC(運営)の株式を所有しているオーナー企業であるハズブロの企業価値に匹敵する金額のため、場合によっては会社が倒産するレベル問題です。

それを踏まえると、ハズブロないし、WotCがそのリスクとこれまでMTGを支えてきた古参ユーザーを裏切って再録禁止を解除するインセンティブはないように思えます。

## 結論

ここまでは数字的な側面を考えてみたが、

つまるところMTGが今後もカードゲームとして人気を維持出来るか？というところがMTGのカードが資産として残るか？という点においての最重要なポイントであり、

それについては自分が実際にプレイしたり、カードを集めたりした感覚から、既存のTCGの中では最も永く遊ばれる可能性が高いと考えている。

また世界最古のTCGであるという歴史的側面やこれまでのプロセス(運営の失敗→復活、近年の統率者人気やIPコラボ)など少しづつ変化をしながらも30年以上続いてきたコンテンツに対する信頼はそれなり可能、というのが僕の見解です。

ということで僕はビットコイン同様に、MTGの再録禁止カードをこれからも集め続けたいと思います。
